เรื่อง: ขอความร่วมมือเพิ่มลิงก์เปิดระบบ Dental Implant จาก iMed

สวัสดีครับทีม iMed

ทางเราต้องการให้ระบบ iMed สร้างลิงก์สำหรับเปิดหน้าเพิ่มคนไข้ใหม่ในระบบ Dental Implant
โดยส่งข้อมูลคนไข้ผ่าน URL Parameter เพื่อให้พนักงานไม่ต้อง copy-paste ข้อมูลเอง


รูปแบบ URL
===========

https://<โดเมน>/patients/new?NameFL=<ชื่อ-นามสกุล>&PatNum=<HN>&PatientGenderMF=<เพศ>&Birthdate_yyyyMMdd=<วันเกิด>


รายละเอียด Parameter
=====================

1. NameFL
   - รูปแบบ: ข้อความ (URL-encoded)
   - ตัวอย่าง: สมชาย+ใจดี
   - คำอธิบาย: ชื่อ-นามสกุลคนไข้

2. PatNum
   - รูปแบบ: ข้อความ
   - ตัวอย่าง: HN12345
   - คำอธิบาย: Hospital Number

3. PatientGenderMF
   - รูปแบบ: M หรือ F
   - ตัวอย่าง: M
   - คำอธิบาย: เพศ (M = ชาย, F = หญิง)

4. Birthdate_yyyyMMdd
   - รูปแบบ: ตัวเลข 8 หลัก
   - ตัวอย่าง: 19820126
   - คำอธิบาย: วันเกิด (ปีค.ศ.+เดือน+วัน)


ตัวอย่าง URL จริง
==================

https://<โดเมน>/patients/new?NameFL=%E0%B8%AA%E0%B8%A1%E0%B8%8A%E0%B8%B2%E0%B8%A2+%E0%B9%83%E0%B8%88%E0%B8%94%E0%B8%B5&PatNum=HN12345&PatientGenderMF=M&Birthdate_yyyyMMdd=19820126

หมายเหตุ: ชื่อภาษาไทยต้อง URL-encode ก่อนใส่ใน URL
ภาษาโปรแกรมส่วนใหญ่มีฟังก์ชันนี้อยู่แล้ว เช่น
- JavaScript: encodeURIComponent()
- Python: urllib.parse.quote()
- C#: Uri.EscapeDataString()
- Java: URLEncoder.encode()


สิ่งที่จะเกิดขึ้น
==================

เมื่อพนักงานกดลิงก์นี้ ระบบ Dental Implant จะเปิดหน้าเพิ่มคนไข้ใหม่
โดยเติมข้อมูลจาก URL ให้อัตโนมัติ พนักงานแค่ตรวจสอบแล้วกดบันทึก


หมายเหตุเพิ่มเติม
==================

- ไม่จำเป็นต้องส่งครบทุก parameter ส่งเฉพาะที่มีก็ได้
- โดเมนจะแจ้งให้ทราบอีกครั้งเมื่อยืนยันแล้ว
